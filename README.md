# 打工人假期日历
这是一个uniapp小程序，可以编译成各个终端应用。具体参考uniapp介绍。

### 功能列表：
- 自定义日历组件，展示阳历、农历、假期标记、调休标记
- 展示最近一年的所有假期，标记下一个假期的剩余天数
- 黄历、 宜、忌、喜神、福神、财神、贵神、彭祖、冲煞
- 最最重要：本程序的所有展示数据完全前端JS计算，不依赖任何外部服务端接口
